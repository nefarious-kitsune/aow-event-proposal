# EXP 點數規則

[English](exp-point-rules)

| 難度       | EXP 點數 | 比較             |
| ----------- | -----------| ---------------------- |
| 簡單        | 25         | 取得四星稀有士兵有**24**點 |
| 尚可        | 50         | 取得一級傳奇英雄有**60**點 |
| 一般        | 100        | 將傳奇英雄從7升級到8有**480**點 |
| 難          | 200        | 將傳奇英雄從10升級到11有**600**點 |
| 很難        | 400        | 將傳奇英雄從14升級到15有**980**點 |
| 非常難      | 800        | 將傳奇英雄從15.3升級到16有**1140**點 |
